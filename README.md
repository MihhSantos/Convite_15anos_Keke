## 📱 Sobre o projeto

Este projeto foi desenvolvido como um convite digital moderno, onde o usuário interage com a tela como se estivesse abrindo um envelope real.

A aplicação permite:

* Abrir o convite de forma interativa
* Confirmar presença via WhatsApp
* Visualizar sugestões de presente
* Acessar localização via Google Maps
* Navegar entre telas com experiência fluida

---

## 🎯 Funcionalidades

✔ Abertura interativa do convite
✔ Botões posicionados sobre imagem (UX mobile)
✔ Integração com WhatsApp
✔ Integração com Google Maps
✔ Sugestão de presente com opção PIX
✔ Navegação entre telas
✔ Música de fundo durante a experiência

---

## 🛠 Tecnologias utilizadas

* HTML5
* CSS3
* JavaScript (Vanilla)
* LocalStorage
* Responsividade Mobile

## 🧠 Aprendizados

Durante o desenvolvimento, foram trabalhados conceitos importantes como:
